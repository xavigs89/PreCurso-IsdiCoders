APUNTES! https://www.codecademy.com/learn/learn-html/modules/learn-semantic-html/cheatsheet El HTML semántico se refiere a la práctica de utilizar etiquetas HTML de manera que el código refleje la estructura y el significado del contenido, en lugar de simplemente su apariencia visual. En otras palabras, se trata de escribir HTML de una manera que sea más descriptiva y significativa, facilitando la comprensión tanto para los desarrolladores como para los navegadores y otros agentes que procesan la información. La introducción de etiquetas semánticas en HTML5 ha mejorado significativamente la capacidad de los desarrolladores para estructurar sus documentos web de manera más significativa.
